【メニューSE追加プラグイン Ver.1.01】
http://hororo.wp.xdomain.jp/198/

2018/02/26更新 v470対応版

-------------------------------------------------------------------------------------------------
■機能
-------------------------------------------------------------------------------------------------
　「メニューボタン（歯車）」「メニュー画面の各ボタン」「セーブ・ロード領域」
　「メニュー、セーブ、ロード、バックログのcloseボタン」にSEを指定するプラグインです。
　　
　◆出来ない事
　　・ボタン毎に個別に設定はできません。
　　・追加したボタンには対応できません。
　　・class名を変更した場合は動作しません。
　　・本体改造をしてないので、若干微妙な所もあります。
　
　◆微妙な点
　	・「タイトルに戻る」はアラートが出るのでクリック時の動作がオカシイです。
　	　※OK→クリック音しない、Cancel→クリック音（環境によります）
　	・セーブ・ロードデータにカーソルON時SEを設定した場合、クリック後表示切替えですぐON時のSEが鳴る。

-------------------------------------------------------------------------------------------------
■使い方
-------------------------------------------------------------------------------------------------
　①data/others/plugin に 「menu_se」を入れてください。
　
　②効果音ファイルはプロジェクトフォルダのsoundフォルダに入れてください。
　
　③first.ks からプラグインを呼び出してください。
　　[plugin name="menu_se"]
　　
　④パラメータで使用する効果音ファイルを指定してください。

　◆パラメーター（※不要な場合は省略可）
　　・メニューボタン（歯車アイコン）
　　　menubutton_clickse    //ボタンをクリックした時
　　　menubutton_enterse    //ボタンの上にマウスカーソルが乗った時
　　　menubutton_leavese    //ボタンの上からマウスカーソルが外れた時

　　・メニュー画面のボタン
　　　menu_item_clickse     //ボタンをクリックした時
　　　menu_item_enterse     //ボタンの上にマウスカーソルが乗った時
　　　menu_item_leavese     //ボタンの上からマウスカーソルが外れた時

　　・メニュー/セーブ/ロード/バックログの閉じるボタン（共通）
　　　menu_close_clickse    //ボタンをクリックした時
　　　menu_close_enterse    //ボタンの上にマウスカーソルが乗った時
　　　menu_close_leavese    //ボタンの上からマウスカーソルが外れた時

　　・セーブデータ
　　　save_list_clickse     //ボタンをクリックした時
　　　save_list_enterse     //ボタンの上にマウスカーソルが乗った時
　　　save_list_leavese     //ボタンの上からマウスカーソルが外れた時

　　・セーブ/ロード/バックログのスマホ用スクロールボタン
　　　button_smart_clickse  //ボタンをクリックした時


　◆記述例
　　例：メニュー画面のボタンにSEを指定する場合。
　　　[plugin name="menu_se" menu_item_clickse="decision5.ogg" menu_item_enterse="decision3.ogg" menu_item_leavese="cancel2.ogg"]
　　　
　　　※パラメータが多くて分りにくい場合は、init.ks を直接編集した方が簡単かもしれません。
　　
　◆他のセーブ・バックログ・メニュー関係のプラグインと併用する場合
　　①このプラグインのinit.ks から [sysview] を削除してください。（htmlファイルは削除してOK）
　　②併用するプラグインで使用するhtmlの一番下に、下記コードを追記してください。
　　　<script type="text/javascript" src="./data/others/plugin/menu_se/menu_se.js">
　　　
　　※試してないので上手く動かない場合はご了承ください…（汗
